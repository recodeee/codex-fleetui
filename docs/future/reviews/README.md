# Protocol Review Files

This directory archives bi-weekly review notes for `docs/future/PROTOCOL.md`.
Reviews are the mechanism that keeps the protocol from drifting from
reality.

## Cadence

- Every 14 days.
- Captain-on-call rotates each review.
- Output: one review file under this directory, named `YYYY-MM-DD.md`.

## Process

1. Run `bash scripts/protocol/protocol-state.sh --table` and paste output.
2. Walk new, promoted, demoted, and stale entries.
3. Rotate captains where needed.
4. Capture open questions and action items.
5. Commit and link from the next PR touching `docs/future/`.
